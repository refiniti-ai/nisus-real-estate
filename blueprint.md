# Project Blueprint

## Overview

This project is a modern, responsive real estate website. It showcases property listings in an engaging and interactive way, incorporating 3D models and advanced search functionalities. The design is clean, professional, and visually appealing, with a focus on user experience.

## Design and Style

*   **Layout:** Responsive and mobile-first, using CSS Grid and Flexbox for flexible and adaptive layouts.
*   **Color Palette:**
    *   Primary: `#1a202c` (dark blue-gray)
    *   Secondary: `#2d3748` (medium gray)
    *   Accent: `#48bb78` (green)
    *   Text: `#f7fafc` (off-white)
*   **Typography:**
    *   Headings: "Montserrat", sans-serif
    *   Body: "Roboto", sans-serif
*   **Iconography:** Feather Icons for clean and modern icons.
*   **Effects:**
    *   Subtle animations on scroll and hover to enhance user interaction.
    *   Parallax scrolling effects for a sense of depth.
    *   Glassmorphism effect for semi-transparent elements.

## Features

### Implemented

*   **Header and Navigation:** A sticky header with a navigation menu that smoothly scrolls to different sections of the page.
*   **Hero Section:** A full-screen hero section with a background image and a prominent call-to-action.
*   **Property Listings:** A grid of property cards with images, prices, and key details.
*   **3D Model Viewer:** An interactive 3D model of a featured property using Three.js.
*   **Agent Profiles:** A section with photos and contact information for real estate agents.
*   **Contact Form:** A functional contact form for inquiries.
*   **Footer:** A footer with social media links and copyright information.
*   **Responsive Design:** The layout adapts to different screen sizes, from mobile phones to desktops.
